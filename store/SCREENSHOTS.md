# Screenshot Checklist

Use this checklist to gather publish-ready screenshots before submitting to the Connect IQ Store.

## Suggested Device Set

- Small watch: `fr55`
- AMOLED watch: `fr265` (or `venu3`)
- Small cycling computer: `edge130`
- Mid-size cycling computer: `edge840`

## Suggested Screenshot Set

Capture at least one screenshot per display class.

1. Before first lap (field shows `--`)
2. After first completed lap (example numeric zone value like `4.7`)
3. During interval workout with multiple laps completed

## Naming Convention

- `fr55-last-lap-max-zone.png`
- `fr265-last-lap-max-zone.png`
- `edge130-last-lap-max-zone.png`
- `edge840-last-lap-max-zone.png`

## Capture Steps (Simulator)

1. Run the app on the target simulator device.
2. Start an activity profile where the data field is visible.
3. Trigger lap events until the value is displayed.
4. Save screenshot from simulator UI and place files under `store/screenshots/`.

## Final Review Before Upload

- Text is legible at arm's length
- Value is not clipped at any edge
- Contrast is clear in bright and dim themes
- Screenshots show realistic workout context
