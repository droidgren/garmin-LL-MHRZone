# Permissions and Privacy

Use this text in the Connect IQ Store submission form.

## Permissions Used

### `UserProfile`

Required to read the user-configured heart rate zone boundaries so the data field can compute and display the last lap maximum HR zone correctly.

No other permissions are used.

## Privacy Text (Store Policy)

Last Lap Max HR processes activity heart rate data and heart rate zone settings locally on the Garmin device to compute the maximum heart rate zone for the last completed lap.

The app does not transmit, store externally, sell, or share personal data.

The app does not require network access, user accounts, or cloud services.

## Data Handling Summary

- Data collection: none outside normal on-device activity metrics
- Data transmission: none
- Third-party sharing: none
- Retention outside device: none
