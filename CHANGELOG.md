# Changelog

All notable changes to this project are documented in this file.

## 1.0.1 (2026-03-09)

- Added a setting to enable or disable the `LL Max HRZ` header text.
- Default behavior keeps the header enabled.
- Updated README and release notes for the new setting and version bump.

## 1.0.0 (2026-03-05)

- Initial release of Last Lap Max HR.
- Displays the maximum heart rate zone of the last completed lap as a decimal value.
- Uses profile-based heart rate zone boundaries from User Profile.
- Optional Garmin-style HR zone digit colors with an on-device toggle (default ON).
- Broad device-target support across watches, cycling computers, and outdoor handhelds.
